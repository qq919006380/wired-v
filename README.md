# reverse-string-package

这个简单的 npm 包提供了一个功能，用于反转给定的字符串。

## 安装

使用 npm 安装这个包：

```bash
npm install wired-v

yarn add wired-v

```js
import { reverseString } from 'wired-v';

const myString = "Hello, World!";
const reversed = reverseString(myString);

console.log(reversed); // "!dlroW ,olleH"
